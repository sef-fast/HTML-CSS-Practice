# E-commerce Product Card Series
A growing collection of product card UI components
built with pure HTML & CSS.

## Product Card 01 — Nike Air Zoom
![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

**Where it's used:** E-commerce stores, sneaker shops, retail websites

**Problem it solves:** Displays product info cleanly with price,
discount badge, and call-to-action button in minimal space

**Techniques used:**
- CSS Custom Properties (Variables)
- Position absolute for discount badge
- Image zoom effect on hover
- Smooth card lift transition
- Google Fonts (DM Sans)
- fit-content width on category tag

## Product Card 02 — Dark Luxury Watch
![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

**Where it's used:** Luxury e-commerce, watch stores,
high-end fashion websites

**Problem it solves:** Presents premium products with a dark,
elegant design that builds trust and urgency with dual badges,
star ratings, and dual action buttons

**Techniques used:**
- Advanced CSS Variables for full dark theme
- backdrop-filter blur on stock badge
- Gradient overlay using inset: 0
- Dual badge system (promo + stock)
- Star rating UI with filled/empty states
- Two Google Fonts combined (Playfair Display + DM Sans)
- flex: 1 for equal button sizing
- Image zoom on hover
